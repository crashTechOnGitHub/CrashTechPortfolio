# CrashTechPortfolio
CrashTech Portfolio site developed on Vue js Framework

## How to Run on local?
#### 1) Clone Repo to local 
#### 2) Go to file location of package.json
#### 3) Open cmd at that location
#### 4) run ``` npm install ```
#### 5) run ``` npm start ```
#### 6) hit http://localhost:3030/ url on web browser
